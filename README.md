About The Project
========
* React Notepad
========
Built With MERN Stack:
Mongo
Express
React
Node
Tailwind CSS

Getting Started
========
* npm
npm install npm@latest -g
I am currently on node v16.20.0 (Run node -v from the terminal)

Installation
========
* Clone the repo
git clone git@github.com:seandavidrobbins/react-notepad.git

* Install NPM packages
npm i

* Start the server
npm run dev
